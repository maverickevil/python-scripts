## Blog系统

### 集成
```bash
# 查看数据库表结构
cat ${root}/blog/static/blog.sql

# SQL导入
mysql -h 127.0.0.1 -P 3306 -u root -p'Admin@12345' blog < ${root}/blog/static/blog.sql

# 导入sdk环境
pip install -r ${root}/blog/requirements.txt
```
